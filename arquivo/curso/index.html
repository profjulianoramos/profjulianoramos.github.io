O curso

Este curso vai além de um preparatório para a Certificação Linux LPI. Ele possui um escopo voltado para a formação de profissionais, prontos para o mercado de trabalho.

O curso não é somente para quem pretende fazer a prova de certificação LPI. Ele também é fundamental para quem deseja obter um conhecimento técnico e de boas práticas em Administração de sistema Linux.

Além da videoaula, você terá acesso a um ebook exclusivo para o curso, com mais de 180 páginas e um simulado para testar o seu conhecimento.

Sumário do curso, tudo o que você irá aprender:

  • Comandos GNU e Unix
  • Manipulação de arquivos de configuração
  • Gerenciamento de arquivos e diretórios
  • Redirecionamento de comandos
  • Gerenciamento de processos
  • Pesquisar arquivos de texto com expressão regular
  • Editor VI
  • Arquitetura do sistema
  • Gerenciamento de blockquote
  • Configuração de Gestor de Boot
  • Gerenciamento de bibliotecas
  • Instalação e configuração de softwares Debian
  • Instalação e configuração de softwares Red Hat
  • Gerenciamento de permissões de usuários e grupos
  • Criação e alteração de hard link e soft link
  • Localização de arquivos no linux
  • Customização do ambiente Shell
  • Lógica de programação - Aprendendo a programar no linux
  • Criação de Shell Script
  • Gerenciamento de banco de dados SQL
  • Configuração de interface do usuário GUI
  • Administração de contas de usuários
  • Automatizar e agendar tarefas administrativas
  • Criação de servidor NTP
  • Gerenciamento de arquivos de LOG
  • Gerenciamento de e-mail local
  • Configuração de impressoras remotas
  • Fundamento de rede de computadores
  • Configuração básica de rede
  • Resolução de problemas de rede
  • Cliente DNS
  • Segurança e boas práticas
