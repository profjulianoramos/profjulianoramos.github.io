Curso ao-vivo LPIC-1 101-500

O curso


Este curso vai além de um preparatório para a Certificação Linux LPI. Ele possui um escopo voltado para a formação de profissionais, prontos para o mercado de trabalho. O curso não é somente para quem pretende fazer a prova de certificação LPI. Ele também é fundamental para quem deseja obter um conhecimento técnico e de boas práticas em Administração de sistema Linux.

Além da videoaula, você terá acesso a um ebook exclusivo para o curso, com mais de 180 páginas e um simulado para testar o seu conhecimento.

Sumário do curso, tudo o que você irá aprender:

  • Comandos GNU e Unix
  • Manipulação de arquivos de configuração
  • Gerenciamento de arquivos e diretórios
  • Redirecionamento de comandos
  • Gerenciamento de processos
  • Pesquisar arquivos de texto com expressão regular
  • Editor VI
  • Arquitetura do sistema
  • Gerenciamento de blockquote
  • Configuração de Gestor de Boot
  • Gerenciamento de bibliotecas
  • Instalação e configuração de softwares Debian
  • Instalação e configuração de softwares Red Hat
  • Gerenciamento de permissões de usuários e grupos
  • Criação e alteração de hard link e soft link
  • Localização de arquivos no linux
